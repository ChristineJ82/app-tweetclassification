Philipphines Typhoon tutorial

Hi! Thanks a lot for helping us in classifying the tweets collected on Typhoon Pablo.

The application is really simple. It loads a tweet and asks you to classify the links that you could find in it.

There are 4 possible categories:

  • Picture: link points to picture(s) of general damage/flooding
  • Video: link points to video(s) of general damage/flooding
  • Location: tweet/picture/video includes reference to location, place, town etc.
  • Other: link does not point to picture(s) of general damage/flooding

If you find a link, click on it in and it will open itself in a different tab or window, check the information in the tab/window and see in which category fits. If the link is related to one of the previous categories, you will have to paste the link in input box.

If you find a reference to a location, you will be able to place it in a map

Usually you will:

  • find the name of the city
  • paste it in the input field and
  • click in the search button Search

The map will search that location and add a blue marker, then you have to place a marker in the position that the tweet is referencing.

To place the marker:

  • click in the button: Add damage/impact marker
  • hover the mouse to place it at the location (a green typhoon marker will appear)
  • click to set the coordinates
Tip: If you need to fix the position you can drag and drop it again.

Once you have chosen one of the categories, click in the Submit classification button and you will have classified one tweet.

That's all!
